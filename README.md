# AssaultJunkies

Simple internal cheetos for fun, made for the best game `AC`.

## Build

- [Cmake](https://cmake.org/)
- [Msvc](https://visualstudio.microsoft.com/pt-br/vs/features/cplusplus/)

```
cd folder
cmake -S . -B build
cmake --build build --config Debug|Release --target ALL_BUILD -j 14

```
